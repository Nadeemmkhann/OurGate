OurGate!

Visitors record

Our Society Management System is a web application that will automate , manages and maintains all the record for visitor in an online mode The entries of visitor are entered by the admin and the approve or reject pop-up will be generated on the user login Flat user can pre entries the visitors for approval from the supervisor

E-Notice board

E-Notice Board in which admin or Supervisor will announce the event or function that will be going to take place in society. Flat user can see the announcement from their respective login and can comment on the announcement

Automatic Generate Maintenance Bill

Our system will have the functionality of generating the maintenance bill automatically and bill will be mailed to flat user

Payment Gateway For Maintenance Bill

Payment gateway will be provided to the flat user to pay the maintenance bill in online mode

Complaint Box

Complaint box or issue box will be provided to all the user login
